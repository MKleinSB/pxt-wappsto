# wappsto:bit

Micro:bit extension for wappsto:bit using NB-IoT

```blocks
Wappsto.sendNumberToWappsto(1, 1, WappstoTransmit.OnChange);
Wappsto.sendStringToWappsto("string", 16, WappstoTransmit.ASAP);
```

## Reference



## Howto build
* npm install pxt
* pxt target microbit
* pxt install
* make

output is located in built-directory
